Detailed Description

template<GLenum B, class I>
class gale::wrapgl::Bindable< B, I >

This is a wrapper class implementing the Curiously Recurring Template Pattern (CRTP) for all objects that can be bound to the OpenGL state.

The template argument B is the name to query the binding and I refers to the implementation class, which needs to provide the following static methods: createObject(), destroyObject(), setCurrent(), hasSameType() and optionally getCurrent().
