Detailed Description

template<typename T>
class gale::math::HMatrix4< T >

Homogeneous matrix class implementation based on column vectors.

The matrix is stored as 4x4 numbers organized in column-major order in memory as required by OpenGL, i.e. the matrix entry at row r, 0 <= r <= 3, and column c, 0 <= c <= 3, is located at offset i = c*4 + r. For a nice article about column vs. row vectors see http://fgiesen.wordpress.com/2011/05/04/row-major-vs-column-major-and-gl-es/.

OpenGL example usage:

Quatf q(Vec3f::Z(),45.0f*Constf::DEG_TO_RAD());
HMat4f m=q.getMatrix();
glMultMatrixf(m);
