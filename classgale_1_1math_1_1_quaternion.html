Detailed Description

template<typename T>
class gale::math::Quaternion< T >

Quaternion implementation based on a scalar for the real number part and a vector for the imaginary number part.

It has common linear algebra and math operations specialized on computer graphics.

OpenGL example usage:

Quatf q(Vec3f::Z(),45.0f*Constf::DEG_TO_RAD());
HMat4f m=q.getMatrix();
glMultMatrixf(m);

Performs a cubic Bezier interpolation between the quaternions q and r based on a scalar s within the quadrilateral defined together with the "tangent" quaternions a and b (the curve touches the midpoint of the "line" from a to b when s = 0.5).

Optionally, an interpolator may be specified. For performance reasons, s is not clamped to [0,1].
