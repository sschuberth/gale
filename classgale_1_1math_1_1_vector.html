Detailed Description

template<unsigned int N, typename T>
class gale::math::Vector< N, T >

Vector class implementation based on a tuple, featuring common linear algebra and math operations specialized on computer graphics.

OpenGL example usage:

glBegin(GL_LINE_LOOP);
glVertex3fv(Vec3f::X());
glVertex3fv(Vec3f::Y());
glVertex3fv(Vec3f::Z());
glEnd();
