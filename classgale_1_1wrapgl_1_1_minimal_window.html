Detailed Description

This class creates minimal window suitable for OpenGL rendering.

Creates a window with the specified title, a client size of the given width and height, an OpenGL context with the attributes in context_attr attached to it and a pixel format that matches pixel_attr.

The window will be hidden initially, but it will become the current render context.
