Detailed Description

template<unsigned int N, typename T, class C>
class gale::math::TupleBase< N, T, C >

Tuple base class implementation with N elements of type T.

The Curiously Recurring Template Pattern (CRTP) is used so that a derived class' operators have the proper argument and return type.
