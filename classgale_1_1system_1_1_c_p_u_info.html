Detailed Description

Using the x86 CPUID instruction, this class reports detailed processor capabilities.

For details consult these documents:

  • Intel Processor Identification and the CPUID Instruction, see http://www.intel.com/Assets/PDF/appnote/241618.pdf.
  • AMD CPUID Specification, see http://support.amd.com/us/Processor_TechDocs/25481.pdf.
