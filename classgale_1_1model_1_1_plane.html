Detailed Description

Plane class implementation based on the equation required by glClipPlane():

1) A*x + B*y + C*z + D = 0

Here, A, B, C are the components of the normal vector, and D is the negative distance to the plane for any point x, y, z on the plane. Note that in the Hessian Normal Form (HNF), D is changed in sign so it becomes the positive distance to the plane: 

2) A*x + B*y + C*z = D

While internally equation 1) is used, all public methods assume the HNF, i.e. equation 2) to be used, as it is more natural to deal with positive distances.

OpenGL example usage:

Plane p(Vec3d(0,1,-1),0.5);
glClipPlane(GL_CLIP_PLANE0,p);
