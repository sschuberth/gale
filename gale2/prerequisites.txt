In order to create the build system and compile GALE, you will need the
following software to be installed on your development machine.


Common
======

- CMake 2.6.4, not newer (http://www.cmake.org/)
- Doxygen 1.6.0 or newer (http://www.doxygen.org/)


Windows
=======

Tools
-----

- msysGit, that is "Git for Windows" (http://msysgit.googlecode.com/)
- PHP, needs to be in PATH (http://www.php.net/)

Compiler
--------

- Visual Studio 2005 or Visual Studio 2008 (the Express editions should do)


Linux
=====

Tools
-----

- GNU BASH (http://www.gnu.org/software/bash/)
- GNU Coreutils (http://www.gnu.org/software/coreutils/)
- GNU Findutils (http://www.gnu.org/software/findutils/)
- PHP (http://www.php.net/)
- Sed (http://sed.sourceforge.net/)
- Wget (http://www.gnu.org/software/wget/)

Compiler
--------

- GCC 4.1 or newer (http://gcc.gnu.org/)
