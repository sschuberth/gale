Detailed Description

A collection of interpolation (and approximation) methods.

For a nice introduction, see http://altdevblogaday.org/2011/03/21/moving-beyond-the-linear-bezier/.

Piecewise linear interpolation of the values in v at position s.

The position in range [0,1] is mapped to the array size. If closed is true, the values are treated as being periodic, resulting in a closed curve. Also see http://local.wasp.uwa.edu.au/~pbourke/miscellaneous/interpolation/.

inlinestaticprivate

Returns the indices i0 and i1 for position s into an array with maximum index n (the position in range [0,1] is mapped to the array size).

If closed is true, the array is treated as being periodic, resulting in the indices to wrap. The return value equals the position scaled to range [0,1] relative to the interval.
