Detailed Description

template<typename T>
struct gale::math::ColorChannel< T >

Helper class with only a single template parameter.

This is needed because partial specialization of class templates by only specifying the differing member functions (not the whole class) is not supported by C++ for multiple template arguments. Using this class we can specialize only those class template member functions that should not be implicitly obtained from the base template.
