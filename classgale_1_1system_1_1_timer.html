Detailed Description

Simple timing class with resume capability.

Constructor & Destructor Documentation

gale::system::Timer::Timer ( )
inline

The default constructor resets the timing, so there no need to call reset() manually.

On Windows, this also locks the current thread to the first CPU for compatibility.

References reset(), s_instances, and s_mask.
